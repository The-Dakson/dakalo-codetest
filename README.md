## ✨ Features

### 👨‍🏫 Admin Features
- **Test Management** – Create, edit, delete, and open/close tests
- **Submissions** – View all student submissions with code preview
- **Auto-Grading** – AI-powered grading using Claude API
- **Manual Grading** – Assign scores and feedback (0-100)
- **Public/Private Grades** – Choose visibility per submission
- **Polls** – Send Yes/No polls to all students in real-time
- **Announcements** – Broadcast messages or send private comments
- **Student Messages** – View and reply to student questions
- **Live Presence** – See which students are currently online
- **PDF Reports** – Download graded submissions as PNG/PDF
- **ZIP Downloads** – Download submitted code files

### 🧑‍🎓 Student Features
- **Test Taking** – Code editor with syntax highlighting and line numbers
- **File Upload** – Submit ZIP files for coding challenges
- **Timer** – Countdown with auto-submit on expiration
- **Attempt Limits** – Configurable submissions per test
- **Grade Viewing** – See scores and instructor feedback
- **Leaderboard** – Public grades shown in ranked order
- **Polls** – Vote on instructor questions with ability to change
- **Messaging** – Send up to 3 messages per day to instructor
- **Real-time Replies** – See instructor replies instantly
- **Live Online** – See other active students
